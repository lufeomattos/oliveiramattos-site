# Política de Privacidade

**Última atualização:** 17 de Fevereiro de 2026

## 1. INTRODUÇÃO

O escritório **Oliveira Mattos e Moraes e Silva Advogados Associados** ("Escritório", "nós", "nosso") reconhece a importância da proteção de dados pessoais de todos os terceiros que se relacionam conosco, sejam eles nossos clientes, parceiros ou simples usuários de nosso website ("Usuários", "você").

Esta Política de Privacidade ("Política") descreve como e por que coletamos, armazenamos, usamos e compartilhamos seus dados pessoais. Ela também explica os direitos que você tem em relação aos seus dados e como pode exercê-los.

Esta Política foi elaborada em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD) - Lei nº 13.709/2018.

Caso, após a leitura deste documento, você ainda tenha questionamentos, poderá entrar em contato conosco através do e-mail: **contato@oliveiramattos.adv.br**.

## 2. COLETA DE DADOS PESSOAIS

Coletamos seus dados pessoais das seguintes formas:

### 2.1. Dados fornecidos diretamente por você

| Grupo de Dados | Forma de Coleta | Quais dados são coletados |
| :--- | :--- | :--- |
| **Identificação** | Formulário de Contato | Nome, e-mail, telefone |
| **Comunicação** | Formulário de Contato | Mensagem, área de interesse |

### 2.2. Dados coletados automaticamente

| Grupo de Dados | Forma de Coleta | Quais dados são coletados |
| :--- | :--- | :--- |
| **Navegação** | Cookies e Logs | Endereço IP, tipo de navegador, páginas visitadas, tempo de permanência |
| **Localização** | Endereço IP | País, estado, cidade |

## 3. USO DE DADOS PESSOAIS

Utilizamos seus dados pessoais para as seguintes finalidades:

- **Prestação de serviços jurídicos:** Para clientes que nos contratam.
- **Comunicação:** Para responder a suas dúvidas e solicitações.
- **Melhoria do site:** Para entender como você usa nosso site e melhorá-lo.
- **Marketing:** Para enviar newsletters e informações sobre nossos serviços (com seu consentimento).
- **Segurança:** Para proteger nosso site e nossos usuários.

## 4. BASES LEGAIS PARA TRATAMENTO

O tratamento de seus dados pessoais é realizado com base nas seguintes hipóteses legais:

- **Consentimento:** Quando você nos autoriza a tratar seus dados (ex: formulário de contato).
- **Execução de contrato:** Para cumprir nossas obrigações contratuais com você.
- **Obrigação legal:** Para cumprir com obrigações legais e regulatórias.
- **Interesse legítimo:** Para atender a interesses legítimos do Escritório, sem ferir seus direitos e liberdades.

## 5. COMPARTILHAMENTO DE DADOS PESSOAIS

Não compartilhamos seus dados pessoais com terceiros, exceto nas seguintes situações:

- **Parceiros e fornecedores:** Para viabilizar nossos serviços (ex: provedor de hospedagem).
- **Obrigações legais:** Para cumprir com ordens judiciais ou requisições de autoridades.
- **Proteção de direitos:** Para proteger os direitos do Escritório em caso de litígio.

## 6. DIREITOS DOS TITULARES

Você, como titular de dados pessoais, tem os seguintes direitos:

- **Confirmação da existência de tratamento:** Saber se tratamos seus dados.
- **Acesso aos dados:** Obter uma cópia dos seus dados.
- **Correção de dados:** Solicitar a correção de dados incompletos, inexatos ou desatualizados.
- **Anonimização, bloqueio ou eliminação:** Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos.
- **Portabilidade:** Solicitar a portabilidade de seus dados para outro fornecedor.
- **Eliminação dos dados:** Solicitar a eliminação de seus dados tratados com seu consentimento.
- **Informação sobre compartilhamento:** Saber com quem compartilhamos seus dados.
- **Informação sobre a possibilidade de não consentir:** Saber as consequências de não fornecer seu consentimento.
- **Revogação do consentimento:** Revogar seu consentimento a qualquer momento.

Para exercer seus direitos, entre em contato conosco através do e-mail: **contato@oliveiramattos.adv.br**.

## 7. SEGURANÇA DOS DADOS

Adotamos medidas de segurança técnicas e administrativas para proteger seus dados pessoais, incluindo:

- Criptografia de dados em trânsito (HTTPS)
- Acesso restrito a dados pessoais
- Monitoramento contínuo de nossos sistemas
- Backups regulares

## 8. RETENÇÃO DE DADOS

Manteremos seus dados pessoais apenas pelo tempo necessário para cumprir as finalidades para as quais foram coletados, ou para cumprir com obrigações legais.

## 9. COOKIES

Nosso site utiliza cookies para melhorar sua experiência. Cookies são pequenos arquivos de texto que são armazenados em seu dispositivo. Você pode gerenciar suas preferências de cookies através das configurações do seu navegador:

- [Internet Explorer](https://support.microsoft.com/pt-br/help/17442/windows-internet-explorer-delete-manage-cookies)
- [Mozilla Firefox](https://support.mozilla.org/pt-BR/kb/ative-e-desative-os-cookies-que-os-websites-usam)
- [Google Chrome](https://support.google.com/accounts/answer/61416?co=GENIE.Platform%3DDesktop&hl=pt-BR)
- [Safari](https://support.apple.com/pt-br/guide/safari/sfri11471/mac)
